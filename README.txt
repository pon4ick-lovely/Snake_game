# Snake Game

A simple snake game built using Python and the Pygame library.

## Installation

To run this game, you'll need to have Python and Pygame installed on your system. You can install Pygame using pip:

```bash
pip install pygame

Usage
To play the game, simply clone this repository and run the snake_game.py file:
python snake_game.py

The game will start and you can control the snake using the arrow keys. Eat the food to grow the snake and avoid colliding with the walls or your own body.

Controls
Arrow keys: Move the snake
Q: Quit the game
C: Continue after losing
